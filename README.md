# Einleitung

Lightstories ist eine Homepage für Fotographie. Der Fotograph präsentiert auf der Homepage seine Arbeiten für Kunden. Kunden sollen primär die Möglichkeit haben, Buchungsanfragen für Fotoshootings zu stellen.

Der Fotograph Thorsten Kolb konzentriert sich auf die folgende 3 Schwerpunkte:

- Hochzeitsfotographie
- Portraitfotographie
- Familienfotographie

## Anforderungen

Es soll für den Fotographen einfach sein, die Texte auf der Homepage zu verändern, die Bilder auszutauschen oder neue Bilder hinzuzufügen sowie neue Artikel zu schreiben.

Die Homepage soll auch einfach durch eine KI verstanden, betreut und gewartet werden können.

Veränderungen sollen immer einfach lokal getestet werden können (auch durch die KI selbst - playwright) und wenn alle Änderungen OK sind soll die Homepage zur Produktion deployed werden.
